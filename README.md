# Yandex Music Psychograph Bot

Телеграм-бот, который принимает ссылку на **публичный** плейлист Яндекс Музыки, вытягивает список треков и выполняет психографический анализ по Big Five с помощью LLM.

## Docker

- Создайте `.env` на основе [.env.example](./.env.example):

```bash
docker compose down -v
docker compose up -d
docker compose logs -f bot
```

## Использование

Отправьте боту ссылку вида:
```
https://music.yandex.ru/users/<user>/playlists/<kind>
```

Бот:
- извлечёт до `MAX_TRACKS` треков (исполнитель — трек),
- покажет превью,
- выполнит психографический анализ на основе вашего промпта,
- вернёт структурированный ответ.

## Примечания

- Плейлисты должны быть **публичными**. Для приватных используйте `YM_TOKEN` (опционально).
- Для экономии токенов LLM длинные плейлисты обрезаются; остаток будет указан в сообщении.
- Вы можете заменить модель и провайдера LLM, адаптировав вызов в `analyze_with_llm`.
